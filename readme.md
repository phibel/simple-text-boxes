## Description

This Wordpress plugin provides two shortcodes:

```
[text_boxes]
  [text_box]TEXT1[/text_box]
  [text_box]TEXT2[/text_box]
  [text_box]TEXT3[/text_box]
  [text_box]TEXT4[/text_box]
[/text_boxes]
```

### How to install
